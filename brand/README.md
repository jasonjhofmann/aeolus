# Brand assets

The `brands` quality-scale rule requires icon/logo PNGs submitted to the
[`home-assistant/brands`](https://github.com/home-assistant/brands) repository
(not bundled here). Needed before claiming the rule:

- `icon.png` — 256×256, square, transparent background
- `icon@2x.png` — 512×512
- `logo.png` / `logo@2x.png` — optional, horizontal
